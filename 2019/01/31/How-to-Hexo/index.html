How to Hexo

2019-01-31

1. Membuat Akun dan Repository Github

Buat akun Github.

New Repository -> isikan nama repository wajib dengan nama : (usernama-kalian).github.io -> set Public -> Create repository.

2. Install Node.Js

Jadi pastikan terlebih dahulu apakah kalian sudah menginstall nodejs dan npm atau belum. kalian bisa mengechecknya diterminal dengan command

Jika sudah, kalian bisa meng-check Hexo kalian dengan command

Jika sudah akan muncul versi Hexo kalian.

3. Membuat Github Pages

4. Memilih theme

Buka website Hexo Theme dan download salah satu tema yang kalian inginkan
Buka Atom dan buka script _config.yml
Pada bagian theme, isikan dengan nama tema kalian.
